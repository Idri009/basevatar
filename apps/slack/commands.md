# Slack Commands

## `/review`

-   **Description**: Use the `/review` command to approve or reject an image.
-   **Usage**: `/review (Approve or Reject) #image-id`
-   **Example**: `/review Approve 12345`

---

## `/vote`

-   **Description**: Use the `/vote` command to add a new voting option. You can specify a day number optionally.
-   **Usage**: `/vote (Theme or Color) (#xxxxx, #yyyyy or Books) [dayNumber]`
-   **Example**: `/vote Color #ff0000,#00ff00 54`
-   **Note**: If the day number is not specified, the vote will be set for the next day.

---

## `/output`

-   **Description**: Use the `/output` command followed by a day number to get the output for a specific day.
-   **Usage**: `/output #dayNumber`
-   **Example**: `/output 5`

---

## `/image`

-   **Description**: Use the `/image` command followed by `#imageId` to get the image.
-   **Usage**: `/image #imageId`
-   **Example**: `/image 12345`

---

## `/user`

-   **Description**: Use the `/user` command followed by `#walletId` to retrieve information about a specific user.
-   **Usage**: `/user #walletId`
-   **Example**: `/user 67890`

---

## `/settings`

-   **Description**: Get current settings.
-   **Usage**: `/settings`

---

## `/today`

-   **Description**: Get today's images.
-   **Usage**: `/today`
